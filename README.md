# Final - Node + Express scaffold

Minimal Node.js + Express scaffold created for development.

Prerequisites
- Node.js 16+ installed

Install and run

```powershell
npm install
npm start
```

Development

```powershell
npm run dev
```

Project layout
- `src/index.js` - server bootstrap
- `src/app.js` - express app and middleware
- `src/routes` - example routes
- `src/middleware` - error handler

Notes
- Add other routes/controllers as needed.
